# Didjest — Zero-Code Университет

Одностраничный сайт (landing page) в стиле [Zerocoder.ru](https://zerocoder.ru) — первый университет зерокодинга в СНГ.

## Инструменты

| Инструмент | Назначение |
|---|---|
| **HTML5** | Семантическая разметка страницы |
| **CSS3** | Стилизация, анимации, адаптивная вёрстка |
| **Vanilla JS** | Интерактивные элементы (чистый JavaScript без библиотек) |
| **Google Fonts (Inter)** | Основной шрифт |

## Особенности проекта

- **Тёмная тема** с фиолетово-розовыми градиентами
- **Градиентный текст** с анимацией shimmer (бегущий градиент)
- **Floating-карточки** с параллакс-анимацией в hero-секции
- **Стеклянная навигация** (glassmorphism) с backdrop-filter
- **Аккордеон FAQ** с плавным раскрытием
- **Анимированные счётчики** статистики (появляются при скролле)
- **Reveal-анимации** — блоки плавно появляются при прокрутке
- **Адаптивный дизайн** — корректное отображение на всех устройствах (десктоп, планшет, мобильные)
- **Burger-меню** для мобильной версии

## Структура секций

1. **Hero** — главный экран с призывом к действию
2. **Partners** — логотипы партнёров
3. **Courses** — карточки курсов (Нейросети, Веб-дизайн, Zero-Code)
4. **About** — информация об университете и статистика
5. **How it works** — 4 шага обучения
6. **Reviews** — отзывы студентов
7. **CTA** — форма подписки на практикум
8. **FAQ** — частые вопросы
9. **Footer** — подвал с контактами и ссылками

## Запуск

Открой `index.html` в любом современном браузере. Для работы не требуется сервер или сборщик.
